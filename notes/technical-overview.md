This repo is a personal notebook and blogging platform using Hugo with the Anubis theme. It’s minimalist and responsive, designed to jot down ideas and reflections quickly.

### Key Features
- Responsive design with dark mode
- Pagination, taxonomies, and social sharing
- Supports Disqus, Isso, Utteranc.es comments
- Google Analytics for tracking
- Math typesetting with KaTeX

### Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/justin-napolitano/bbc.git
   cd bbc
   ```

2. Add the Anubis theme as a submodule:
   ```bash
   git submodule add https://github.com/mitrichius/hugo-theme-anubis.git themes/anubis
   ```

3. Run locally:
   ```bash
   hugo server -D
   ```
   Navigate to `http://localhost:1313` in your browser.

### Gotchas
Make sure you have the extended version of Hugo installed for proper functionality.
